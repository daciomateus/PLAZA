# Plaza Beach Club

Landing page comercial em React + Tailwind construída a partir do material atual do PDF e das imagens disponíveis no workspace.

## Estrutura

- `src/App.jsx`: composição principal da landing page
- `src/data/content.js`: conteúdo textual e CTAs de WhatsApp
- `src/components/`: componentes reutilizáveis
- `src/styles.css`: estilos globais com Tailwind
- `assets/images/`: imagens reaproveitadas do material disponível

## Como rodar

```bash
npm install
npm run dev
```

Para gerar build de produção:

```bash
npm run build
```
